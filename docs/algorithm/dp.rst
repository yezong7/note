动态规划
========

背包问题
--------

0-1 背包
^^^^^^^^

- 每个物品只能选一次
- ``dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])``

完全背包
^^^^^^^^

- 每个物品可以选无限次
- ``dp[i][j] = max(dp[i-1][j], dp[i][j-w[i]] + v[i])``

子序列问题
----------

最长递增子序列 (LIS)
^^^^^^^^^^^^^^^^^^^^

- ``dp[i]`` 表示以 ``nums[i]`` 结尾的 LIS 长度
- ``dp[i] = max(dp[j] + 1)`` for all ``j < i`` and ``nums[j] < nums[i]``

最长公共子序列 (LCS)
^^^^^^^^^^^^^^^^^^^^

- ``dp[i][j]`` 表示 ``s1[:i]`` 和 ``s2[:j]`` 的 LCS 长度

经典题目
--------

- **509** - 斐波那契数
- **70** - 爬楼梯
- **300** - 最长递增子序列
- **1143** - 最长公共子序列
